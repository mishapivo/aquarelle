== Description ==
Minimal Travelogue is a clean and modern WordPress Child Theme of Galway Lite. The theme was designed with the “mobile first” design, meaning it was designed for mobile from the beginning and was adapted for bigger screen afterward. This result in a beautiful experience for the users, on all devices.


WordPress theme "Minimal Travelogue" is a child theme of "galway-lite".
galway-lite Theme is licensed under the GPL3.
