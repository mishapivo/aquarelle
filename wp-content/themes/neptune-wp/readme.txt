A Wordpress theme for small busineses 

HOW TO SET UP
Download Advanced Custom Fields and Kirki Plugin (Highly Recomended)
1.create a page and assign it the "home lite" template under attributes.
2. Go to settings > reading > static page and select the page you just created
3. Go back to the page you created in step 1
4. Add content in the new custom metaboxes
5. remember to install kirki and ACF (included in TGM) in order to use all the theme's features.
