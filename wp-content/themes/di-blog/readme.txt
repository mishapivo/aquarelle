== Description ==

Di Blog is a Responsive, SEO Friendly, Customizable and Powerful WordPress Theme for Creatives and Blogger. All Google Fonts, Footer widgets, More than responsive, page builder ready, woo commerce ready and much more.

== Installation == 

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in 'Di Blog' in the search form and press the 'Enter' key on your keyboard.
3. Click on 'Install' and then 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel and customize the taste.
